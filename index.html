About Us

Curriculum

Week 1: Introduction to HTML

Introduction to the basic structure of web pages using HTML, including tags, elements, and attributes.

Week 2: CSS Fundamentals

Exploration of CSS syntax, selectors, properties, and values to style and layout web pages.

Week 3: Responsive Web Design

Learn how to create responsive web designs that adapt to different screen sizes and devices.

Week 4: JavaScript Basics

Introduction to JavaScript programming language, covering variables, data types, functions, and basic DOM manipulation.

Week 5: Advanced CSS

Explore advanced CSS techniques such as flexbox, grid layout, animations, and transitions.

Week 6: JavaScript and the DOM

Dive deeper into JavaScript and learn how to interact with the Document Object Model (DOM) to create dynamic web pages.

Week 7: Front-End Frameworks

Introduction to popular front-end frameworks like React or Vue.js, and building interactive user interfaces.

Week 8: UI/UX Design Principles

Learn about user interface (UI) and user experience (UX) design principles, wireframing, and prototyping.

Week 9: Web Accessibility

Understanding the importance of web accessibility and implementing accessible design and development practices.

Week 10: Web Performance Optimization

Optimizing web page performance by reducing load times, leveraging caching, and optimizing assets.

Week 11: Version Control with Git

Using Git for version control, including branching, merging, and collaborating on projects.

Week 12: Final Project

Apply the skills learned throughout the bootcamp to develop and present a complete web design project.
